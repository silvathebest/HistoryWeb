Добро пожаловать в увлекательное игровое web-приложение "Найдите лишнее"

О программе
  • Приложение разработанно с применением JavaScript HTML CSS
  • Была использована библиотека bootstrap4
  • Все сцены подобраны и отредактированы с помощью adobe Photoshop
Инструкция
  • Найдите все лишние объекты
  • Когда найдете - кликните по лишнему объекту левой кнопкой мыши
  • После того как найдете все объекты, вы можете перейти на следующую сцену нажатием на левую кнопку мыши по свободному пространству на сцене
  • Вы можете вернуться на прошлую сцену, нажатием на левую часть изображения
